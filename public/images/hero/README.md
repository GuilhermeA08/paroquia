# Imagens do Hero

Esta pasta deve conter as imagens de fundo para a seção hero do site.

## Arquivos sugeridos:
- church-bg.jpg - Imagem de fundo principal (fachada ou interior da igreja)

## Formato recomendado:
- Resolução: 1920x1080px ou superior
- Formato: JPG
- Tamanho máximo: 300KB (otimizada para web)
